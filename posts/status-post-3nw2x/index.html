Express JS - Set Up 404 Error Page

NodeJS Tech - 27 May, 2021

Here is the Code to Setup 404 Error page in Express.js.

app.use('/', function(req, res) {
    res.status(404).json({
        error: 1,
        message: '404 Not Found'
    });
});
